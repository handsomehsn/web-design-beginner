# web-design-beginner

Küçük ama etkileyici bir responsive web sitesi projesi.

Bu proje, modern web tasarımının temel bileşenlerini gösterir:

- Responsive tasarım
- Flexbox ve Grid düzenleri
- Etkileşimli iletişim formu
- Footer alanı
- CSS ile görsel düzen ve button efektleri
- JavaScript ile dark-light mode,form verisi kaydetme ve hesap makinesi

## Proje Hakkında

`index.html`, `style.css` ve `script.js` dosyalarından oluşan basit bir portföy / tanıtım sayfasıdır. Proje, hem masaüstü hem de mobil cihazlarda iyi görünmesi için tasarlanmıştır.

## Nasıl Görünmeli

GitHub üzerinde proje sayfanızda şu bilgilerin yer alması faydalıdır:

- Proje adı ve kısa tanımı
- Neler yaptığınız ve hangi teknolojileri kullandığınız
- Görseller veya canlı önizleme açıklaması (isteğe bağlı)
- Projeyi çalıştırmak için gerekli adımlar

## Çalıştırma

1. Bu depoyu bilgisayarınıza klonlayın:
   ```bash
   git clone https://github.com/kullaniciadi/web-design-beginner.git
   ```
2. Klasöre girin:
   ```bash
   cd web-design-beginner
   ```
3. `index.html` dosyasını tarayıcınızda açın.

## Canlı Önizleme (GitHub Pages)

Bu projeyi GitHub Pages ile hızlıca yayınlayabilirsiniz:

1. Depoyu GitHub'a yükleyin (örneğin `main` dalı):
   ```bash
   git add .
   git commit -m "Add project files"
   git push origin main
   ```
2. GitHub deposu sayfasında `Settings` -> `Pages` bölümüne gidin.
3. `Source` olarak `main` dalını ve `/ (root)` klasörünü seçin, kaydedin.
4. Birkaç dakika içinde sayfanız `https://<kullaniciadi>.github.io/<repo-adi>/` adresinde erişilebilir olacaktır.

Not: Eğer `main` yerine `gh-pages` dalı kullanmak isterseniz, `gh-pages` dalı oluşturup içeriği oraya deploy edebilirsiniz.

## Özellikler

- `Hero` bölümü
- `About` bölümünde kartlı içerik
- `Services` ızgara düzeni
- `Projects` bölümü ve demo içerikleri
- `Contact` formu, form doğrulama ve mesaj kaydetme
- `Footer` alanında iletişim bilgisi ve isim kredisi
- Dark / light mod desteği
- Google Fonts `Poppins` entegrasyonu
- Hesap makinesi demo projesi
- Klavye ve buton desteği ile hesap makinesi yazma ve silme
- Yumuşak kaydırma (smooth scroll) navigasyon
- LocalStorage ile tema ve adres kaydetme

## Yapılanlar

Bu projede aşağıdaki geliştirmeler yapıldı:

- Dark mode ve gündüz modu arasında geçiş ekleme
- Dark mode için sayfa arka planları, kartlar ve başlık renklerini düzenleme
- `Google Fonts` bağlantısını `head` bölümüne ekleme
- Hesap makinesine geri silme (`backspace`) ve klavye desteği ekleme
- `Başla` butonunu `İletişim` bölümüne kaydırma olarak ayarlama
- `contactSubmissions` kullanarak gönderilen form verilerini localStorage'da saklama
- Footer notuna hem geliştirici ismini hem de GitHub Copilot adını ekleme

## Yapılması Muhtemel İşler

Bu projede şu ek çalışmalar yapılabilir:

- Gerçek proje sayfalarına yönlendiren canlı bağlantılar eklemek
- `Footer` bölümüne sosyal medya ikonları ve bağlantıları eklemek
- Görseller veya proje ekran görüntüleri ekleyerek projeyi daha görsel hale getirmek
- Performansı artırmak için CSS/JS küçültme ve resim optimizasyonu yapmak
- Erişilebilirlik için form etiketlerini ve kontrastı iyileştirmek
- Responsive menü tasarımını mobilde daha belirgin yapmak
- `Projects` bölümüne daha fazla proje veya demo içeriği eklemek
- Bazı animasyonlar ve hover efektleri ile kullanıcı deneyimini zenginleştirmek

Bu proje GitHub Copilot tarafından destekli olarak yazılmıştır.
